# Spectra Obfuscator bot

bot discord can be used to obfuscating your own lua script's, built in with
discord js and main source of this obfuscator [ Prometheus ](https://github.com/prometheus-lua/Prometheus)
you can use that source to obfuscate your script using CLI from them if u want.

# How to use?
- First, clone this repository using `git clone https://github.com/ak4eska/obf-bot.git`
- After that, enter into the directory using `cd obf-bot`
- run `npm i` or `npm install` to installing all dependesies

# How to configure and run the bot?
to configure the bot you can renamed first .env.example to .env, and insert the token and app id of your own bot
example:
```
TOKEN=MTQ1....
CLIENT_ID=123....
```
oh i forgot to say to you, you need the install node.js first (version 14+ at least)
you can't build your own bot or token or finding app id?

After you configure the bot, you need to run this commands `node index` or `node index.js`
